# CCM Construction Website

Modern, responsive website for CCM Construction - a home improvement contractor serving Central New Jersey.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **React Router DOM** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework

## Features

- Responsive design (mobile-first)
- Modern UI inspired by leading contractor websites
- Service showcase with image cards
- Service area coverage map
- Customer testimonials carousel
- Contact form with service selection
- SEO-optimized meta tags

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
